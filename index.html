DFW Sights & Landmarks

Visit Project Web Page

A single-page web application, built using the Knockout.js framework. The webpage app displays a Google Map of an area and pin various points of interest that contain Sights and landmarks in the DFW, Texas area. Users can view all landmarks in the map and display them. When a sight is selected, additional information about a landmark is presented in the popup Window. This application use SQL to store the sights infomation and request them. The application also use Google maps API and and Yahoo Weather API services.

Requierments For The Project:

  • Javascript, Knockout.js, Google Maps API, Yahoo Weather API, HTML, CSS, Bootstrap
    • Documentation: Github
    • Date: June 2017
    • Service: Web Development
    • Visit Web Page: DFW Map
